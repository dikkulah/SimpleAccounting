Ft Teknoloji Practium Case
# Frontend - Vaadin Framework ile kodlandı.
- Frontend uygulaması çalışmaz ise mvn komutu çalıştırılmalı.
```maven
mvn clean package
```
- Veritabanlıarını ayağa kaldırma
```
docker-compose up -d
```
